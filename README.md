# NeoFNC — Finanças Inteligentes

Site completo de controle financeiro pessoal com design **dark glassmorphism**, pronto para hospedar no **GitHub Pages**.

## ✨ Funcionalidades

### Design
- 🌑 Dark moderno com **Glassmorphism**
- ✨ Animações suaves em cards e modais
- 📱 Responsivo para celular e desktop
- 🎨 Gradientes e efeitos de profundidade

### Dashboard
- 💰 Cards de **Saldo Total, Receitas, Despesas, Resultado**
- 📈 Gráfico de **Linha** — evolução mensal de receitas vs despesas
- 🍕 Gráfico de **Pizza (Doughnut)** — distribuição de gastos por categoria
- 🪙 **Ticker de Crypto** em tempo real (BTC, ETH, USD/BRL via API pública)
- 📋 Lista das últimas transações

### Transações
- Cadastro de **Receitas, Despesas e Transferências**
- Filtro por mês com chips interativos
- Categorias: Moradia, Alimentação, Transporte, Saúde, Lazer, Educação, Salário, Investimento, Assinaturas, Outros
- Atualização em tempo real dos valores ao adicionar

### Contas
- Múltiplas contas bancárias
- Saldo calculado automaticamente com base nas transações
- Cores personalizadas por conta

### Cartões de Crédito
- Controle de limite e fatura
- Barra de uso visual com alerta de cores (verde → amarelo → vermelho)
- Dia de vencimento e bandeira

### Metas Financeiras
- Progresso visual com barra animada
- Prazo configurável
- Ícones temáticos

### Relatórios
- Receitas e despesas por categoria com barras de progresso
- Gráfico de barras com evolução dos últimos 6 meses

## 💾 Armazenamento

Dados salvos no **LocalStorage** do navegador (sem servidor, 100% privado e offline).

## 🛠 Tecnologias

- HTML5 + CSS3 + JavaScript puro
- **Chart.js 4.4** para os gráficos
- Google Fonts (Outfit + JetBrains Mono)
- APIs públicas: CoinGecko (cripto) + AwesomeAPI (câmbio)
